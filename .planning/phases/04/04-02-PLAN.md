# Current lease detail page pattern

The lease detail page at `src/app/(dashboard)/leases/[id]/page.tsx` already shows:
- Unit number + tenant name + status badge
- Lease period, monthly rent, unit type, total collected
- Payment history table
- Renew/Terminate action buttons

The page already has `useAuth()` and is protected by `<AuthGuard>`. It uses:
- `fetch()` for data (will be replaced by PocketBase SDK in Phase 3, but tenant link generation is a simple write)
- `toast` from `sonner` for feedback
- `Modal` component for confirmations
- `Badge` from `@/components/ui/badge` for status

Token generation needs to use `crypto.randomBytes(16).toString('hex')` for 32-char hex strings. This is available in Node.js and browsers (via Web Crypto API: `crypto.randomUUID()` or `crypto.getRandomValues()`).

For the PocketBase SDK write:
- `pb.collection('leases').update(leaseId, { tenantAccess: newToken })`
- This is a simple update — no expansion needed

The shareable link format is:
- `/tenant/portal?token={leaseId}:{token}`
- Landlord can copy this to clipboard

**Add functions:**
```typescript
// Generate a new tenant access token
async function generateTenantLink() {
  setLinkLoading(true)
  try {
    const token = crypto.randomUUID() // 36-char UUID (simpler than randomBytes hex)
    // Update lease record via PocketBase SDK
    const pb = await import('@/lib/pocketbase')
    await pb.default.collection('leases').update(id, { tenantAccess: token })
    setTenantLink(`${window.location.origin}/tenant/portal?token=${id}:${token}`)
    setShowLinkModal(true)
    toast.success('Tenant link generated')
  } catch {
    toast.error('Failed to generate tenant link')
  } finally {
    setLinkLoading(false)
  }
}

// Regenerate token (invalidates old link)
async function regenerateTenantLink() {
  setLinkLoading(true)
  try {
    const token = crypto.randomUUID()
    const pb = await import('@/lib/pocketbase')
    await pb.default.collection('leases').update(id, { tenantAccess: token })
    setTenantLink(`${window.location.origin}/tenant/portal?token=${id}:${token}`)
    setShowLinkModal(true)
    toast.success('Tenant link regenerated — old link invalidated')
  } catch {
    toast.error('Failed to regenerate tenant link')
  } finally {
    setLinkLoading(false)
  }
}
```
